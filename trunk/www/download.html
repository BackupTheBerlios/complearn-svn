Pre-Compiled Binaries and Packages

Windows precompiled binaries

ncd and maketree binaries may be downloaded from here and external compressors, such as bzip2 and lzma, may be found here.

Debian and Ubuntu binaries (.deb)

are available for direct download from here for Debian , or here for Ubuntu , however it is easiest to fetch these packages automatically using apt-get.

Debian or Ubuntu apt-get auto-installation

All files are available for automatic download by adding the following two lines to your /etc/apt/sources.list file for Debian:

deb http://debian.cilibrar.com/debian/ etch main
deb-src http://debian.cilibrar.com/debian/ etch main

or these two for Ubuntu:
deb http://ubuntu.cilibrar.com/ubuntu/ dapper main
deb-src http://ubuntu.cilibrar.com/ubuntu/ dapper main

Once these are added, an apt-get update will enable easy installation with just one command:

apt-get install complearn-tools libcomplearn0-dev

To try the 3D demo / toy just use this to grab the many graphical libraries necessary to run it:

apt-get install complearn-gui

Please note that these .deb's are unsigned, and so you must confirm installation of the unauthenticated archives when prompted. You must also allow the apt-get system to fetch several required dependencies.

Core Dependencies

If you are compiling CompLearn from source, and want to take full advantage of the software's features, we highly recommend you download and install the following software packages:

Package Source Version apt-get command Description
gsl* tar.gz 1.6 apt-get install libgsl0-dev scientific library used to generate matrices
bzip2 tar.gz 1.0.2 apt-get install libbz2-dev compression library used in NCD calculations
zlib tar.gz 1.2.2 apt-get install zlib1g-dev compression library used in NCD calculations
gdbm tar.gz 1.8.3 apt-get install libgdbm-dev database library used for Google caching
xml2 tar.gz 2.6.16 apt-get libxml2-dev required csoap dependency
csoap tar.gz 1.0.4 n/a protocol library required for Google queries

* required dependency
